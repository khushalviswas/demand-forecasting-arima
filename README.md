# Demand Forecasting using ARIMA

This project demonstrates a time-series demand forecasting workflow using the ARIMA model.

## Objective

To predict future product demand using historical structured dataset trends for improving supply chain planning.

## Workflow Steps

- Data preprocessing and cleaning
- Trend and seasonality analysis
- ARIMA-based forecasting
- Model evaluation and prediction interpretation

## Tools Used

- Python
- Pandas
- NumPy
- Statsmodels
- Jupyter Notebook

## Outcome

Developed forecasting logic capable of identifying demand trends and supporting inventory planning decisions.

This workflow was explored as part of experimentation during IBM Datathon participation.
